# AI Model Inference API with FastAPI
A secure and scalable AI inference API built using FastAPI that performs sentiment prediction and includes authentication, rate limiting, monitoring, and testing.

This project demonstrates how to build a production-style backend system with machine learning integration and monitoring tools.

Project Features

FastAPI based REST API

AI model inference using Scikit-learn

JWT authentication for secure access

Password hashing using bcrypt

Rate limiting to prevent API abuse

Asynchronous API endpoints

Database integration

Prometheus metrics monitoring

Grafana dashboard visualization

API testing using Postman and Pytest
